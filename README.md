tender
======

A tender color scheme for Vim and its ecosystem

Features:

- dark background
- tender colors
- minimal interface
- extra colorization for languages and plugins:
  - javascript
  - yaml
  - NERDTree
  - GitGutter
  - pangloss/javascript
- coming soon:
  - CSS
  - HTML
  - json
  - vimScript
  - Python
  - PHP
  - markdown
